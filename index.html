Controle de Caixa

Projeto Profissional

Sistema web de controle financeiro desenvolvido para resolver um problema real na empresa onde trabalho. O processo de fechamento de caixa era feito manualmente e apresentava erros frequentes; o controle de cheques pré-datados era uma planilha desorganizada com perda de dados. Desenvolvi uma solução integrada que centraliza tudo em um único sistema, sem necessidade de servidor ou instalação.

  • Fechamento de caixa diário com autenticação por usuário e histórico de sessões.
  • Controle de cheques pré-datados com alertas de vencimento e registro de depósitos.
  • Geração de relatórios formatados para impressão.
  • Dados persistidos no navegador via LocalStorage — sem banco de dados externo.
HTML5 CSS3 JavaScript LocalStorage Autenticação Relatórios
